Loops in javascript

Loops are used to execute a block of code repeatedly until a specified condition is met.

Types of Loops

  • for loop: Executes a block of code a specified number of times.
  • while loop: Executes a block of code as long as a specified condition is true.
  • do...while loop: Similar to the while loop, but guarantees that the block of code will be executed at least once.
  • for...in loop: Used to iterate over the properties of an object.
  • for...of loop: Used to iterate over iterable objects like arrays, strings, etc.
